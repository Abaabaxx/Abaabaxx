我是一名对具身智能充满热情的本科生。在多次国家级机器人竞赛中，我亲手搭建并调试了完整的机器人系统。

 我渴望从一名 **工程师** 转变为一名 **研究者**，探索更鲁棒、更智能的机器人范式。
